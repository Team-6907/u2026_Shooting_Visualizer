### Features

- 3D field visualization
- Live trajectory and impact point
- Shoot-on-move compensation
- Tunable mechanism parameters

### Quick start

```bash
npm install
npm run dev
```

Open: `http://localhost:5173`

```bash
npm run build
npm run preview
```

### Controls

- `WASD` - move robot
- `Space` - stop
- Mouse drag - orbit camera
- Scroll wheel - zoom
